### 2의 제곱 수 - 데이터 양 추정
![data-volume.png](img/data-volume.png)
- 데이터 단위는 2의 제곱수로 표현된다.
- 각 단위들이 2의 몇  제곱인지 어느 정도 규모를 뜻하는지 파악해두면 좋을 것 같다.
